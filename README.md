# StockMarketApp Android
Stock market application with https://www.alphavantage.co/

# Features

- Stock data APIs in JSON, Excel & Google Sheets
- Forex and cryptocurrency data feeds
- Company search feature
- Details of the company
    - Name
    - Industry
    - Country
    - Company history
    - Market summary
